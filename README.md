# Social Action Marche 公式サイト

「私もできる」を見つける1日 — 2026.8.9(日) 潮風公園

1ページ完結のランディングページです。ビルドツール不要で、そのまま公開できます。

## ファイル構成

```
index.html      … ページ本体（全セクションのテキストはここ）
css/style.css   … デザイン（色は :root の変数で一括変更可）
js/config.js    … ★サイト設定（フォームURLなど）
js/main.js      … 動作（メニュー・アニメーション・CTA切替）
```

## よくある更新作業

### 1. 申込みフォームのURLが決まったら

[js/config.js](js/config.js) を開いて、1行変えるだけです。

```js
formUrl: "https://forms.gle/xxxxxxxx",
```

これでサイト内の全CTAボタン（ヒーロー・最終CTA・スマホ追従バー）が
「Coming Soon」から本物の申込みボタンに自動で切り替わります。

### 2. 登壇者・出店団体・キッチンカーを追加したいとき

[index.html](index.html) 内の `<!-- ▼編集: ... -->` コメントが目印です。
既存のカード（`.speaker-card` / `.booth-card`）をコピーして、名前とテキストを書き換えてください。
「Coming Soon」カードは、全員決まったら削除してOKです。

### 3. 写真・ロゴが用意できたら

- 登壇者写真: `.speaker-card__face` の中のSVGを `<img>` に差し替え
- OGP画像: `index.html` の `<head>` 内コメントアウトされた `og:image` を有効化

## 公開方法

静的サイトなので、以下のいずれでも公開できます。

- **GitHub Pages**: リポジトリにpushして Settings → Pages を有効化
- **Netlify / Vercel**: フォルダをドラッグ&ドロップするだけ
- レンタルサーバー: フォルダごとFTPアップロード
